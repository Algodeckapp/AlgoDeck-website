# Complete AlgoDeck Website Build Prompt

## Overview
Create a modern, professional landing page website for **AlgoDeck** - a mobile trading automation platform (Android & iOS app, NOT a web app). The website should clearly communicate that users need to download the mobile app to use the platform.

---

## Design Guidelines

### Brand Identity
- **Name:** AlgoDeck
- **Tagline:** "Automate Your Trading Empire"
- **Color Scheme:**
  - Primary: Deep Blue (#1E40AF / #3B82F6)
  - Accent: Purple (#8B5CF6)
  - Success: Green (#10B981)
  - Warning: Amber (#F59E0B)
  - Danger: Red (#EF4444)
  - Background: Dark navy (#0A0E27)
  - Surface: Lighter dark (#1A1F3A)

### Design Style
- Modern, sleek, professional
- Dark theme with vibrant accent colors
- Clean typography (Satoshi or similar modern sans-serif)
- Gradient accents (blue to purple)
- Smooth animations and transitions
- Mobile-first responsive design
